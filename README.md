# 📄 Proposal Writer Skill（Web3 版）

一个专为 **Web3 行业**从业者设计的 Claude Skill，帮你快速生成专业的中文 Proposal，输出格式兼容 Notion 直接粘贴。

---

## 这个 Skill 是什么

`proposal-writer` 是一个写作辅助工具，封装了 Web3 行业市场营销从业者写 Proposal 的完整框架和逻辑。

它支持两种类型的 Proposal：

**① Pitch Proposal**
用于向投资人、合作方或社区论证一个项目/方向的可行性。
叙事结构：市场背景 → 竞品横向对比 → 我们的运营模式 → 解决了什么痛点

**② 活动策划 Proposal**
用于策划一场 Web3 活动（黑客松、社区聚会、峰会、AMA、快闪等）。
- **对内版**：内部汇报或申请预算，包含成本预估
- **对外版**：招募赞助商或寻求合作方，包含赞助权益和 Package 方案；自动从赞助商视角审视并输出优化版

---

## 能解决什么问题

- 不知道 Proposal 该怎么组织结构 → Skill 内置完整框架，自动引导填写
- 每次写都从零开始，费时费力 → 提供信息即可生成完整初稿
- Pitch 里竞品分析不知道怎么写 → 自动生成对比表格 + 逐项深度分析
- 对内和对外 Proposal 容易混 → 根据用途自动切换模板
- 赞助方案写出来没有说服力 → 从赞助商痛点出发写权益，而不是罗列通用曝光位
- 写完不知道好不好 → 对外版自动从赞助商视角审视初稿，输出优化版作为最终版本
- 写完还要手动排版进 Notion → 输出直接兼容 Notion Markdown，粘贴即用

## 如何使用

### 第一步：安装 Skill

1. 下载本仓库中的 `proposal-writer.skill` 文件
2. 打开 [Claude.ai](https://claude.ai)，点击左下角头像进入设置
3. 进入 **Settings → Capabilities**，确认 **Code execution and file creation** 已开启
4. 进入 **Customize → Skills**，点击 **+** 按钮 → **+ Create skill**
5. 上传 `proposal-writer.skill` 文件，安装完成

### 第二步：开始使用

安装后，直接在对话框里说你想做什么，Skill 会自动触发。

**触发示例：**
- "帮我写一个 Pitch Proposal"
- "我要策划一个线下活动，帮我做活动策划"
- "我需要一份赞助方案"
- "帮我 pitch 一个 Web3 社区运营项目"

### 第三步：回答引导问题

Skill 会根据你的需求，分批问你几个关键问题，比如：
- 这个项目/活动是什么？
- 主要竞争对手有哪些？
- 活动是对内汇报还是对外找赞助？

你不需要一次准备好所有信息，信息不完整时 Skill 会先生成框架，用 `[待补充]` 标注缺失内容。

### 第四步：获取输出

- **Pitch Proposal / 活动策划对内版**：生成完整初稿，可直接复制进 Notion，有需要告诉 Claude 修改哪个部分
- **活动策划对外版（赞助方案）**：自动输出「初稿 → 赞助商视角审视 → 最终优化版」三段式内容，直接用最终版即可

## 输出结构

**Pitch Proposal**
```
01 市场背景
02 竞品横向对比（表格 + 深度分析 + 横向小结）
03 我们的运营模式
04 我们解决了什么
05 团队 & 过往成绩
06 下一步
```

**活动策划 Proposal（对外 / 赞助方案）**
```
01 这是一个什么活动（一段话 + 目标人物画像 + 活动内容 bullet）
02 为什么现在做这个活动（市场背景 + 行业痛点，击中赞助商决策逻辑）
03 我们怎么做（时间线 + 活动执行内容）
04 赞助商可以获得什么（痛点对应权益 + 额外服务 + Package 方案）
05 联系我们
```

**活动策划 Proposal（对内 / 申请预算）**
```
01 这是一个什么活动
02 为什么做这个活动（背景与痛点 + 活动目标）
03 我们计划怎么做（执行模块 + 时间线 + 预计产出）
04 本次活动预估成本
05 申请与下一步
```
